# Anne Isabel W - E-Portfolio Website

A modern, responsive e-portfolio website for Anne Isabel W, a B.Tech CSE student at SRM Institute of Science and Technology.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, modern design with smooth animations and transitions
- **Interactive Navigation**: Smooth scrolling navigation with active link highlighting
- **Dynamic Content**: Animated hero section with typing effect and parallax scrolling
- **Project Showcase**: Detailed project cards with technology tags and descriptions
- **Skills Display**: Organized skill categories with hover effects
- **Education Timeline**: Visual timeline of educational background
- **Resume Download**: Functional resume download feature
- **Contact Section**: Footer with social links and contact information

## Sections

1. **Home**: Hero section with personal introduction
2. **About**: Detailed about section with statistics
3. **Skills**: Programming languages, databases, and technologies
4. **Projects**: Academic and personal projects with descriptions
5. **Education**: Educational background with timeline
6. **Resume**: Downloadable resume section
7. **Contact**: Footer with contact information and social links

## Technologies Used

- **HTML5**: Semantic markup for structure
- **CSS3**: Modern styling with animations and transitions
- **JavaScript**: Interactive features and functionality
- **Font Awesome**: Icons for visual elements
- **Google Fonts**: Inter font for typography

## Key Features

### Navigation
- Fixed navigation bar with smooth scrolling
- Mobile-responsive hamburger menu
- Active link highlighting based on scroll position
- Background change on scroll

### Animations
- Typing animation for hero name
- Parallax scrolling effect
- Intersection Observer for scroll animations
- Hover effects on cards and buttons

### Responsive Design
- Mobile-first approach
- Flexible grid layouts
- Responsive typography
- Touch-friendly interface

## Getting Started

1. Clone or download the project files
2. Open `index.html` in a web browser
3. Navigate through the portfolio sections

## Customization

### Personal Information
Update the following in `index.html`:
- Name and personal details in the hero section
- About section content
- Skills and technologies
- Project information
- Education details
- Contact information

### Styling
Modify `styles.css` to:
- Change color schemes (CSS variables)
- Adjust typography
- Modify animations and transitions
- Update responsive breakpoints

### Functionality
Enhance `script.js` to:
- Add new interactive features
- Modify animations
- Update form validation
- Add new user interactions
